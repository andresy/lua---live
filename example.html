Large-scale learning problems

  • Developement of complex systems

Why not using Matlab?

  • The good side:
    • Fast simple language, with just-in-time-compiler
    • Excellent linear algebra toolbox

  • The bad side:
    • Does not scale: copies everything
    • Not as powerful a true scripting language

Ideal solution:

  • Powerful interpreted scripting language as fast as C
  • Well-designed linear algebra library

Interpreted scripting language

Dynamically typed. Available types:

  • nil
  • boolean (true, false)
  • number (double)
  • string
  • function
  • thread
  • table (hashtable)
  • userdata
